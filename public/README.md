# DND Guru Logo Assets

Included files:

- dnd-guru-logo-original-white-bg.png ,  original preview on white.
- dnd-guru-logo-transparent.png ,  main transparent logo for website headers/hero sections.
- dnd-guru-logo-transparent-square.png ,  square transparent master.
- dnd-guru-logo-2048.png / 1024 / 512 / 256 / 128 / 64 ,  resized PNG exports.
- dnd-guru-logo-1024.webp / 512.webp / 256.webp ,  compressed WebP exports.
- dnd-guru-logo.svg ,  SVG wrapper embedding the logo image. Usable as SVG, but not a hand-vectorized redraw.
- favicon.ico, favicon-16x16.png, favicon-32x32.png ,  browser favicons.
- apple-touch-icon.png ,  iOS home-screen icon.
- android-chrome-192x192.png and android-chrome-512x512.png ,  Android/PWA icons.
- site.webmanifest ,  starter manifest.

Recommended HTML:

```html
<link rel="icon" href="/favicon.ico" sizes="any">
<link rel="icon" href="/dnd-guru-logo.svg" type="image/svg+xml">
<link rel="apple-touch-icon" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">
```

For your main site header, start with dnd-guru-logo-transparent.png or dnd-guru-logo-512.webp.
